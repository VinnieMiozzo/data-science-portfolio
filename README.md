# Data Science & Analytics Portifolio

This repository contains a collection of Data Science and Analytics projects combining Statistical Modeling, Machine Learning, 
Network Science, Time Series Analysis, and Simulation Methods. The goal is to demonstrate applied analytical skills accross different 
domains while mantaining strong methodological rigor and reproducibility.

Some of the projects in the repository are inspired by topics studied during the MIT MicroMasters in Statistics and Data Science, particularly 
in areas such as experimental design, network science, high-dimensional data analysis, and probabilistic modeling.
All projects presented here are independent implementations built using publicly available datasets, expanded with aditional analysis and 
engineering practices to resamble real-world data science workflows.

## About me 

I am a Data Analyst / Data Scientist with experience in statistical analysis, data modeling, and applied machine learning.

My work focuses on transforming complex data into actionable insights through:

- statistical analysis and experimental design
- machine learning and predictive modeling
- data visualization and analytical storytelling
- data modeling and analytical data pipelines

I am particularly interested in the intersection between data science, analytics engineering, and decision-making, where rigorous quantitative methods can support product development, business strategy, and operational improvements.

This repository serves as a technical portfolio showcasing analytical methods and reproducible data science workflows applied across different problem domains. 

## Projects Overview

| Project                                   | Focus                                                        | Key Methods                                                          | Technologies                |
| ----------------------------------------- | ------------------------------------------------------------ | -------------------------------------------------------------------- | --------------------------- |
| Product Analytics & A/B Testing           | Experimental analysis of user behavior and treatment effects | Hypothesis testing, experiment design, causal inference              | Python, Pandas, Statsmodels |
| Network Analysis                          | Graph modeling and similarity analysis in complex networks   | Centrality measures, similarity metrics, graph algorithms            | NetworkX, Numpy, Scipy      |
| Spatial Flow Simulation                   | Modeling dynamic spatial systems and particle movement       | Monte-Carlo simulation, spatio-temporal analysis, Gaussian processes | Numpy, Scipy, Matplotlib    |
| Machine Learning on High-Dimensional Data | Unsupervised learning and classification pipelines           | PCA, t-SNE, clustering, logistic regression                          | Scikit-learn, Pandas        |
